* Setup Keymap

#+begin_src emacs-lisp
(map! :desc "next line"
      :nvi "C-." 'embark-act)
#+end_src

* Setup Ivy / Posframe

This makes the selections use Ivy / Posframe instead of opening a separate buffer

#+begin_src emacs-lisp
(setq embark-prompter 'embark-completing-read-prompter)
(setq embark-indicators nil)
#+end_src
